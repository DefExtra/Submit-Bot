## setup 👨‍💻❓.

1. install [nodeJS](https://nodejs.org/dist/v16.14.2/node-v16.14.2-x64.msi) and [bash](https://git-scm.com/download/win)
2. download the project in you pc
3. edit the `config.def` file
4. click on `run.bat` file
5. enjoy
